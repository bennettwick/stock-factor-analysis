## Critical Conventions
- **FF unit conversion:** Divide `Mkt-RF`, `SMB`, `HML`, `RF` by 100 immediately after loading. yfinance returns are decimal; raw FF data is percent. Mismatched units silently break alpha estimates by a factor of 100.
- **yfinance end date is exclusive.** Use `end="2026-01-01"` to include Dec 31, 2025.
- Use simple returns (`pct_change()`), not log returns.
- Always call `sm.add_constant()` on regression predictors.
- `Mkt-RF` is already an excess return — do not subtract RF from it again.
- Inner-join stock returns with FF factors on date index, then `dropna()`, before any regression or ADF test.
- Rolling windows: 252 days for beta, 60 days for return mean / annualized vol. Annualize vol with `* np.sqrt(252)`.
